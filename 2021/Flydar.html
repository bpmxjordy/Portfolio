Flydar was a brief I set myself over the current lockdown. I created this brief because I have always had a huge interest in aviation and have always noticed that the UI for the radar systems has always looked out of date and a bit confusing.

I didn't want to just create a UI for radar systems, I wanted to create a company that creates and teaches pilots how to use them.
